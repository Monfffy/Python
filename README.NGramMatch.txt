Introduction
--------------
"NGramMatch" is a Python program built for University of Melbourne subject COMP90049-Knowledge Technology  project 1.
It is developed to solve misspelling issue among tweeters.
Package "NGram 3.3" has been used in this program.
Documentation of "NGram 3.3" is avaiable at https://pythonhosted.org/ngram/index.html.
It is a set class that supports lookup by N-gram string similarity.

Environments
-------------
Python 3.4.3
NGram module
Windows 10
