Introduction
--------------
"GED" is a Python program built for University of Melbourne subject COMP90049-Knowledge Technology  project 1.
It is developed to solve misspelling issue among tweeters.
This main function "GED" is built on top of Needleman¨CWunsch algorithm.
Introduction of Needleman¨CWunsch algorithm avaivable at https://en.wikipedia.org/wiki/Needleman%E2%80%93Wunsch_algorithm

Environments
-------------
Python 3.4.3
Windows 10
